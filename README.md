## Run Locally

Clone the project

```bash
  git clone https://github.com/SyedaMahamFahim/Investor-World-React-Landing-Page.git
```

Go to the project directory

```bash
  cd investor-world-react-landing-page
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
